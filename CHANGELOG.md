## 更新记录

### v1.2 — 2026-03-01 | Three.js 3D 预览

**新增功能：**
- 集成 Three.js 3D 渲染引擎（Lazy 加载，不影响首屏性能）
- 顶部栏 2D/3D 一键切换按钮 + 快捷键 `3`
- 3D 模型生成：铝合金框架（金属质感）、半透明玻璃、五金件把手
- 墙体背景 + 窗台，窗户嵌入墙体展示
- OrbitControls 轨道控制（拖拽旋转、滚轮缩放、右键平移）
- 6 个预设视角（正面/背面/左侧/右侧/俯视/透视）+ 平滑过渡动画
- 窗户开启/关闭动画（左开/右开/上悬/推拉）
- 明暗模式切换、墙体显示/隐藏、网格显示/隐藏
- 多窗口并排展示，自动相机适配

**文件变更：**
- 新增 `client/src/components/ThreePreview.tsx`
- 新增 `client/src/lib/window3d.ts`
- 更新 `TopBar.tsx`、`StatusBar.tsx`、`Editor.tsx`
- 更新 `ARCHITECTURE.md`

### v1.1 — 2026-03-01 | 移动端触摸适配

**新增功能：**
- 触摸手势支持：单指绘制/操作、双指捏合缩放、双指平移画布
- 响应式布局：手机/平板自动切换底部工具栏 + 抽屉式属性面板
- 画布右上角浮动缩放控件（+/-/% 按钮）
- iPhone 刘海/底部安全区域适配（viewport-fit=cover + safe-area-inset）
- 所有触摸按钮最小 48x48px，符合移动端可访问性标准

**文件变更：**
- 新增 `client/src/components/MobileToolbar.tsx`

- 新增 `client/src/hooks/useIsMobile.ts`

### v1.0 — 2026-03-01 | 核心编辑器

**新增功能：**
- 2D Canvas 绘图引擎（HTML5 Canvas 2D API）
- 窗口外框绘制（鼠标拖拽 + 网格吸附）
- 中梃/横档分割（递归分格树结构）
- 6 种扇类型配置（固定/左开/右开/上悬/左推/右推）
- 6 种预设窗型模板（固定窗/单开窗/双开窗/三等分窗/推拉窗/上悬窗）
- 工程制图风格尺寸标注
- 报价单生成（CSV 导出 + 打印）
- 撤销/重做系统（最多 50 步历史记录）
- 完整快捷键体系（V/R/M/T/S/H/Del/Ctrl+Z）
- 三栏式专业布局：左工具栏 + 中画布 + 右属性面板
- 工业蓝图美学设计风格

**核心数据模型：**
- WindowUnit → Frame → Opening → Mullion/Sash/Glass 树形嵌套结构
- 型材系列配置（50/60/65/70 系列）

**文件变更：**
- 初始项目搭建，所有核心文件创建
