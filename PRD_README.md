## 3. V5.4 核心变更摘要

| 变更类别 | 具体内容 |
| :--- | :--- |
| **结构性修复** | 修复 EBNF 语法不完整（补充三元/比较/逻辑运算符）、统一主键命名规范（`{entity}_id`）、统一技术栈为 Konva.js + React 19 |

| **下料优化算法** | 新增 BFD + 余料复用算法的完整 Python 伪代码、复杂度分析 |
| **打印模板引擎** | 新增 Handlebars 模板引擎方案、变量体系、布局系统 |

## 4. 如何使用

- **产品经理/项目经理**：应始终维护 `docs/PRD_V5_Complete.md` 的准确性和权威性。任何新的需求或变更，都必须直接更新此文档。
- **开发人员/测试人员**：在开始任何开发或测试工作前，请确保您阅读的是最新版本的 `docs/PRD_V5_Complete.md`。**不要**参考任何 `archive` 目录下的文件。
- **团队新人**：阅读 `ARCHITECTURE.md` 和 `docs/PRD_V5_Complete.md` 是了解项目的最佳起点。
