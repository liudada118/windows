## 4. 如何使用

- **产品经理/项目经理**：应始终维护 `docs/PRD_V6_Complete.md` 的准确性和权威性。任何新的需求或变更，都必须直接更新此文档。
- **开发人员/测试人员**：在开始任何开发或测试工作前，请确保您阅读的是最新版本的 `docs/PRD_V6_Complete.md`。**不要**参考任何 `archive` 目录下的文件。
- **团队新人**：阅读 `ARCHITECTURE.md` 和 `docs/PRD_V6_Complete.md` 是了解项目的最佳起点。
