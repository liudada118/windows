# PRD 画图渲染模块 — 产品文档层面审查笔记

## 已读章节
- 第4章：画布数据 Schema（700-960行）— 完整的 JSON 结构定义
- 第7章：设计门窗画图核心模块（1185-1493行）— 页面布局、工具箱、渲染规则

## 第4章发现的问题

### 4.1 Schema 结构
- ✅ 有完整的顶层字段定义（canvas, frame, mullions, sashes, glasses, grilles, fillings, annotations, metadata, compositeGroup）
- ✅ 每个字段都有 JSON 示例和字段表
- ✅ 有完整的三扇平开窗示例

### 4.2 缺失项
1. **异形框的渲染算法未定义**：frame.type 支持 arch/triangle/trapezoid/circle/polygon，但 PRD 只给了 rectangle 的渲染规则（7.5.1），异形框如何渲染（弧线绘制、三角形填充、多边形裁剪）完全没有定义
2. **弧形中梃的渲染未定义**：Mullion 有 isArc 和 arcHeight 字段，但渲染规则中没有弧形中梃的绘制方法
3. **格条渲染细节缺失**：grilles 有 pattern 类型（grid/diamond/arc/circle/custom），但没有每种 pattern 的具体渲染算法描述
4. **填充物渲染细节缺失**：fillings 有 screen/blind/panel/image 等类型，但没有每种类型的视觉表现定义
5. **customPaths（SVG path 格式）**：格条支持自定义路径，但没有定义 path 的格式规范和渲染方式
6. **compositeGroup 渲染规则缺失**：定义了组合窗的数据结构，但没有定义组合窗在画布上如何渲染（多窗拼接的视觉表现、连接件的渲染）

## 第7章发现的问题

### 7.1-7.4 工具箱和面板
- ✅ 8个Tab工具箱定义完整
- ✅ 右侧面板4个Tab定义完整
- ✅ 交互流程有描述

### 7.5 渲染规则 — 严重不足
1. **只有矩形框的渲染规则**：7.5.1 外框渲染只说了"深灰色3D立体效果"、"45度斜切角"，但：
   - 弧形框怎么渲染？弧线怎么画？
   - 三角形框怎么渲染？斜边的型材怎么表现？
   - 转角框怎么渲染？两个框之间的连接怎么画？
   - 多边形框怎么渲染？
2. **颜色/木纹渲染机制未定义**：7.4.3 颜色Tab定义了纯色和木纹选项，但没有定义：
   - 纯色如何应用到框体（是替换 frameFill？还是叠加？）
   - 木纹纹理如何渲染（是贴图？Canvas pattern？图片填充？）
   - 木纹的方向（水平/垂直/45度）如何处理？
3. **扇的型材渲染未定义**：7.5.3 说"扇框与外框同色，型材面宽更窄"，但没有定义扇框的具体绘制方法（是画一个内缩的矩形框？还是只画开启标记？）
4. **选中状态的完整视觉规格缺失**：
   - 框选中时的视觉效果？
   - 中梃选中时的视觉效果？
   - 扇选中时的视觉效果？
   - 多选时的视觉效果？
5. **拖拽交互的视觉反馈未定义**：
   - 中梃拖拽时的预览效果？
   - 框拖拽移动时的效果？
   - 组件拖入画布时的预览效果？
6. **缩放和平移的渲染行为未定义**：
   - 缩放时文字大小如何变化？（保持固定还是跟随缩放？）
   - 缩放时线宽如何变化？
   - 最小/最大缩放比例？
7. **截面图渲染未定义**：7.5.5 提到"截面图"在底部信息栏，但没有定义截面图怎么画
8. **室内/室外视图切换的渲染差异未定义**：canvas.view 支持 outdoor/indoor，但没有定义切换视图时渲染有什么区别
9. **打印/导出时的渲染规则未定义**：打印和截图时的分辨率、DPI、颜色模式等
10. **高DPI/Retina屏幕适配未定义**
