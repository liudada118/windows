# 画门窗深度竞品分析报告

**版本**: 1.0
**日期**: 2026-03-01
**作者**: Manus AI

---

## 1. 核心结论

经过对"画门窗"（算料版）的深度体验，我发现它是一款高度成熟、功能完备的门窗行业SaaS软件。我们的产品与它相比，**差距是全方位的**，主要体现在以下几个方面：

| 方面 | 画门窗 | 我们的产品 |
|------|--------|------------|
| **产品定位** | 门窗行业一体化解决方案（设计-报价-生产-管理） | 门窗设计工具（画图+3D预览） |
| **核心功能** | 画图、算料、报价、下料优化、订单管理、客户管理、生产管理 | 画图、3D预览、实景融合（初级） |
| **数据驱动** | 完全数据驱动，设计即算料，算料即报价 | 设计与算料脱节，数据模型不完整 |
| **用户体验** | 专业、高效，但学习曲线陡峭 | 简单、直观，但功能缺失严重 |

**一句话总结：我们还在做"玩具"，画门窗已经在做"武器"了。**

## 2. 模块化深度分析

### 2.1 画图模块

画门窗的画图模块是整个软件的核心，其设计理念是"所见即所得，所画即所算"。

**关键功能点**：
- **丰富的模板库**：平开窗、推拉窗、折叠窗、格条等模板数量巨大，覆盖了市面上绝大多数窗型
- **精确的尺寸标注**：多层标注（总尺寸、分格尺寸、扇尺寸），实时更新
- **专业的2D渲染**：3D立体效果的型材渲染、清晰的开启方向标记、五金件图标
- **灵活的交互方式**：模板拖拽、精确放置、尺寸拖拽、键盘快捷键
- **完善的颜色配置**：纯色/木纹分类，可分别设置外框/压线颜色，颜色库可管理

**与我们产品的差距**：
- 我们的模板库太小，扇类型不全，没有格条
- 我们的尺寸标注只有总尺寸，没有分格尺寸
- 我们的2D渲染是简单线条，没有3D效果和五金件图标
- 我们的交互方式单一，缺少精确放置和拖拽功能
- 我们完全没有颜色配置功能

### 2.2 算料与报价模块

这是画门窗的"杀手级"功能，也是我们产品完全缺失的部分。

**核心逻辑**：
1. **型材系列管理**：用户可以自定义型材系列，录入每个型材的截面尺寸、重量、价格、算料公式
2. **公式编辑器**：强大的公式编辑器，支持加减乘除、括号、系统变量，可以定义每个型材的下料长度和数量
3. **自动算料**：画完图后，系统根据选定的型材系列和公式，自动计算出每个型材的下料长度、数量、总重、总价
4. **一键报价**：根据算料结果，自动生成详细的报价单，包含型材、玻璃、五金、加工费、安装费等

**关键发现**：画门窗的算料公式是其核心竞争力，我们获取到的108平开系列公式和系统变量列表是极其宝贵的信息。

### 2.3 下料优化模块

下料优化是门窗厂降本增效的关键环节。

**核心功能**：
- **一维切割优化**：将算料得出的型材下料长度，在原材料上进行最优排布，减少浪费
- **可视化排布方案**：以图形方式展示每根原材料如何切割，一目了然
- **优化参数配置**：可以设置锯缝宽度、原材料长度、余料优先等参数

**与我们产品的差距**：我们完全没有下料优化功能。

### 2.4 3D渲染与实景融合

画门窗的3D渲染和实景融合功能，是其营销签单的重要工具。

**3D渲染**：
- 渲染效果逼真，光影效果好
- 支持更换背景、调节灯光
- 渲染速度快

**实景融合（拍照搭配）**：
- 用户上传现场照片
- 在照片上拖拽放置门窗
- 自动调整门窗的透视和光影，与环境融合
- 技术方案是后端渲染，我们获取到了其renderJson数据结构

**与我们产品的差距**：
- 我们的3D渲染效果一般，缺少光影和背景配置
- 我们的实景融合是纯前端合成，效果不如画门窗的后端渲染自然

## 3. 数据模型分析

通过截取画门窗的renderJson，我们获取到了其核心数据模型。这是一个极其重要的发现，为我们重构产品提供了清晰的指引。

**画门窗数据模型特点**：
- **层级清晰**：Window → Opening → Sash/Mullion/Glass
- **参数化**：所有尺寸、位置、角度都由参数定义
- **配置驱动**：型材、颜色、五金都通过ID关联到配置库
- **算料导向**：数据模型中包含了算料所需的所有信息（型材ID、公式变量等）

**与我们产品的差距**：我们的数据模型过于简单，缺少算料和生产所需的关键字段。

## 4. 总结与建议

画门窗是一款非常强大的竞品，我们与它的差距是系统性的。但我认为，我们有机会通过**差异化竞争**实现赶超。

**我们的优势**：
- **Web端体验**：我们的产品是纯Web应用，免安装，跨平台，体验更轻量
- **交互创新**：我们可以借鉴Figma等现代设计工具的交互理念，做出比画门窗更直观、更易用的画图体验
- **AI赋能**：我们可以利用最新的AI技术，在实景渲染、智能推荐等方面实现弯道超车

**下一步行动建议**：
1. **重构产品定位**：从"画图工具"升级为"AI驱动的门窗设计与营销平台"
2. **补齐核心功能**：按照画门窗的功能列表，分阶段补齐算料、报价、下料优化等核心功能
3. **重构数据模型**：参考画门窗的renderJson，重新设计我们的数据模型，使其能够支撑未来的所有功能
4. **聚焦用户体验**：在画图模块上投入更多精力，做出比画门窗更流畅、更易用的交互体验
5. **发力AI应用**：将AI实景渲染作为我们的核心卖点，做出比画门窗更惊艳的效果
