# 画门窗软件 - 产品功能规格说明书 (PRD)

**版本：** 1.0
**日期：** 2026-03-01
**作者：** Manus AI

## 1. 文档概述

本文档旨在为「画门窗」软件提供一份完整、详细、可直接用于开发的产品功能规格说明。文档覆盖了软件的所有15个功能模块，对每个页面的布局、字段、交互逻辑、业务规则和数据模型进行了精确定义，以确保开发团队能够基于此文档进行高效、准确的软件开发工作。

## 2. 全局功能与约定

### 2.1 顶部导航栏

- **Logo:** 左上角始终显示“画门窗”Logo，点击通常返回首页。
- **右侧菜单:** 右上角包含“小程序”、“教程”链接以及用户头像菜单。

### 2.2 用户头像下拉菜单

点击用户头像后，出现下拉菜单，提供以下功能：

| 菜单项 | 功能描述 |
|---|---|
| 版本信息 | 显示当前用户账号的版本，如“工厂版”。 |
| 升级 | 引导用户至付费升级页面。 |
| 账号信息 | 显示当前登录的账号（手机号）及是否为子账号。 |
| 系统设置 | 跳转至个人中心/系统设置页面。 |
| 修改密码 | 跳转至修改当前账号密码的页面。 |
| 退出登录 | 清除用户会话，返回登录页面。 |

### 2.3 数据表格约定

- **分页:** 所有大数据列表均采用后端分页，底部提供标准的分页控件，包括页码导航和“前往X页”的跳转功能。
- **操作列:** 表格的最后一列通常为“操作”列，包含对单行数据的常用操作按钮，如“详情”、“编辑”、“删除”等。
- **搜索与筛选:** 大部分列表页面在顶部提供基于关键词的搜索框和基于字段（如日期、类型）的筛选器。

## 3. 模块功能详述

### 3.1 首页 (Home)

**URL:** `/home`

#### 3.1.1 页面概述

首页是用户登录后进入的第一个页面，是整个系统的核心枢纽。它主要由三部分构成：顶部导航栏、一级模块快速入口矩阵、以及最近订单列表。

#### 3.1.2 一级模块入口

首页以图标矩阵的形式提供了通往软件所有核心模块的快速入口，共15个模块，分为三大类别。

| 类别 | 模块名称 | 功能简述 |
|---|---|---|
| **通用** | 设计门窗 | 核心功能，用于绘制门窗、阳光房等。 |
| | 订单管理 | 查看、管理所有订单。 |
| | 型材系列 | 管理产品所需的型材、配件等基础数据及算料公式。 |
| | 个人中心 | 实际为教程中心，提供视频教程。 |
| **门店管理** | 炫图 | AI渲染功能，生成门窗在真实场景中的效果图。 |
| | 阳光房 | 独立的3D阳光房设计工具。 |
| | 拍照搭配 | AR功能，将门窗模型叠加到实景照片上。 |
| | 窗型图库 | 预设的窗型模板库，可直接调用。 |
| **工厂管理** | 客户管理 | 维护客户信息，区分成交客户与潜在客户。 |
| | 财务管理 | 管理收款、支出、计件工资等财务数据。 |
| | 车间管理 | 跟踪订单的生产状态（意向单 → 已发货）。 |
| | 仓库管理 | 管理型材、玻璃、五金等物料的库存。 |
| | 计件工资 | 统计员工的计件工资明细。 |
| | 扫码跟踪 | 生产环节通过扫码更新订单状态。 |
| | 设备对接 | 实际跳转至窗型图库，或在教程中提供与CNC设备对接的视频。 |

#### 3.1.3 最近订单列表

首页下半部分展示了最近创建或更新的订单列表，方便用户快速访问。

| 日期 | Date | 2026-03-01 | 订单创建或最后修改日期。 |

### 3.2 订单管理 (Order Management)

#### 3.2.1 订单详情页

**URL:** `/orders/group-detail/{id}`

**页面概述:**
订单详情页是查看和管理单个订单的核心页面，展示了订单的全部信息，包括客户信息、金额、包含的门窗明细、报价信息和操作日志。

**页面结构:**
1.  **顶部操作栏:** 提供“新增”、“优化”、“打印”、“分享”、“更多”等全局操作。
2.  **合同头部信息:** 展示订单的核心摘要信息。
3.  **报价信息面板:** （默认折叠）展示详细的报价条目和金额。
4.  **门窗明细列表:** 以卡片形式逐一展示订单中包含的每一樘门窗。

**合同头部信息字段:**

| 字段名 | 类型 | 描述 |
|---|---|---|
| 合同号 | String | 唯一标识，只读。 |
| 客户 | String | 客户名称和电话。 |
| 地址 | String | 客户地址。 |
| 型材重量 / 玻璃重量 | Number | 自动计算的材料重量。 |
| 总樘数 / 总面积 / 总金额 | Number | 订单的汇总数据。 |
| 已收款 / 未支付 | Number | 订单的财务状态。 |
| 下单日期 / 创建者 | Date / String | 订单的创建信息。 |
| 合同状态 | Enum | 意向单 / 合同单。 |
| 备注 | String | 订单的附加说明。 |

**门窗明细卡片:**

每个卡片代表一樘门窗，包含图纸缩略图、窗号、系列、颜色、尺寸、金额等信息，并提供“编辑”、“打印”、“详情”、“炫图”等针对单樘门窗的操作。

**报价信息面板:**

- 用户可在此添加、编辑、删除报价项，如“基础价格”、“安装费”、“运输费”等。
- 系统会自动计算总金额。
- “基础价格”项通常为所有门窗金额之和，不可手动修改。

### 3.3 设计门窗 (Drawing Board)

**URL:** `/draw`

#### 3.3.1 页面概述

设计门窗是软件最核心的功能模块，提供了一个强大的可视化绘图环境，让用户可以精确地设计各种复杂的门窗、阳光房等产品，并为后续的算料、报价、生产提供数据基础。

#### 3.3.2 页面布局

1.  **顶部工具栏:** 提供全局操作，如“保存”、“撤销”、“恢复”、“设置”、“渲染”、“导出”等。
2.  **左侧工具箱:** 核心的绘图工具集，包含创建门窗基本结构和添加组件的所有工具。
3.  **中央画布:** 无限大的绘图区域，用户在此处进行门窗的设计和编辑。
4.  **右侧属性面板:** 根据当前在画布上选中的对象，动态显示其属性并允许修改。主要包含“订单”、“颜色”、“组件属性”三个标签页。

#### 3.3.3 左侧工具箱详解

| 工具 Tab | 工具名称 | 功能描述 |
|---|---|---|
| **框** | 外框工具 | 提供超过60种预设的外框模板，包括矩形、异形、转角、圆弧等，是所有设计的起点。 |
| **中梃** | 中梃/立柱工具 | 用于在已有的框或格内添加垂直或水平的分割，提供超过30种样式，如固定中梃、活动中梃、带角度中梃等。 |
| **平开扇** | 平开扇工具 | 用于在格内添加各种类型的平开扇，包括内开、外开、上悬、下悬、对开等超过30种模板。 |
| **推拉/折叠** | 推拉/折叠扇工具 | 提供超过12种推拉扇和折叠扇的模板，支持多轨推拉、左右推拉、多扇折叠等。 |
| **格条** | 装饰格条工具 | 用于在玻璃内部添加装饰性的格条，提供超过40种样式，如田字格、菱形格、米字格等。 |
| **填充物** | 填充物工具 | 用于替换玻璃区域为其他填充物，如百叶、平板、玻璃棉等8种类型。 |
| **标线** | 标注工具 | 提供9种标注工具，用于在图纸上添加尺寸、角度、文字等标注信息。 |
| **其他** | 辅助工具 | 包含“合并”、“打断”、“对齐”等6种编辑辅助工具。 |

#### 3.3.4 右侧属性面板

-   **订单 Tab:** 显示当前设计的门窗所属的订单信息，并允许修改窗号、位置等。
-   **颜色 Tab:** 提供一个包含超过20种纯色和8种木纹色的调色板，用户可以为型材、配件指定颜色。
-   **组件属性 Tab:** 这是交互最频繁的面板。当用户在画布上选中任何一个组件（如外框、玻璃、扇、中梃）时，此面板会显示该组件的所有可配置参数，例如：
    -   **外框:** 宽度、高度、型材系列、壁厚等。
    -   **玻璃:** 玻璃类型（如钢化、中空）、厚度、材质等。
    -   **平开扇:** 开启方向、执手位置、五金件型号等。

### 3.4 型材系列与算料 (Product Series & Calculation)

**URL:** `/product-manage/index`

#### 3.4.1 页面概述

型材系列是软件进行成本核算与报价的基础，是系统的核心基础数据。此模块允许企业管理自己的型材、配件、玻璃等物料信息，并为每个系列产品配置复杂的算料公式和设计规则。

#### 3.4.2 型材系列管理

-   **目录结构:** 左侧以树状目录组织所有型材系列，企业可根据品牌、类型等自定义文件夹。
-   **系列卡片:** 右侧以卡片网格形式展示每个具体的型材系列，包含截面图和名称。
-   **核心操作:** 支持对型材系列进行新增、克隆、删除，以及最重要的**“查看公式”**。

#### 3.4.3 系列公式编辑页

**URL:** `/product-manage/product-script/{id}`

这是软件技术含量最高、最复杂的页面，它将门窗设计与后端算料逻辑紧密地结合在一起。

**页面布局:**
-   **左侧绘图区:** 内嵌一个与“设计门窗”模块功能一致的画板，用于绘制测试用的标准窗型。
-   **下方公式配置区:** 核心区域，通过10个Tab页来定义与该系列相关的所有计算规则。

**公式Tab页详解:**

| Tab名称 | 功能描述 |
|---|---|
| **外框/内扇/推拉扇等公式** | 定义不同组件（框、扇）所用型材的长度、数量的计算公式。公式可以使用系统预设变量（如`H`代表高度, `W`代表宽度）和自定义变量。 |
| **产品报价** | 定义该系列产品的销售报价计算公式，可以基于面积、周长、材料成本等多种因素。 |
| **成本核算** | 定义产品的成本计算公式，用于内部核算。 |
| **计件工资** | 定义生产该系列产品时，各工序的计件工资计算方法。 |
| **设计规则** | 定义设计的约束条件，如“最大扇高”、“最小中梃间距”等，画图时会自动校验。 |
| **打孔设置** | 定义型材在CNC加工时的打孔位置、大小、类型等参数。 |

#### 3.4.4 材料管理

**URL:** `/product-manage/sub-bar`

提供一个统一的界面来管理所有基础物料，包括型材、玻璃、五金、配件等。

-   **分类管理:** 可按材料类型（型材、玻璃等）和自定义目录进行分类。
-   **字段定义:** 每种材料都包含编号、名称、颜色、单位、成本、报价、库存警报等详细字段。
-   **批量操作:** 支持从文件导入、批量设置价格等高效操作。

### 3.5 门店与工厂管理模块

#### 3.5.1 客户管理
- **URL:** `/manage/customerManagement`
- 提供标准的CRM功能，包括客户列表、新增/编辑客户、导入/导出等。
- 核心字段包括客户姓名、联系电话、地址、客户类型（成交/潜在）、业务员、关联合同金额等。

#### 3.5.2 财务管理
- **URL:** `/manage/financial`
- 包含概括统计、收款明细、支出明细、计件明细、收款账户五个子模块。
- 提供数据看板、流水记录、报表导出等功能，是企业财务管理的核心工具。

#### 3.5.3 车间管理
- **URL:** `/manage/workshop`
- 以看板形式展示订单在生产环节的状态流转：意向单 → 正式单 → 生产中 → 已完成 → 已发货。
- 管理员可在此页面跟踪所有订单的生产进度，并进行状态变更操作。

#### 3.5.4 仓库管理
- **URL:** `/manage/wareHouse/stockStatus`
- 管理所有物料的库存状态，包括材料管理、领料出库、操作流水、余料管理、仓库设置等功能。
- 与算料公式和生产订单打通，实现库存的自动扣减。

#### 3.5.5 计件工资与扫码跟踪
- **URL:** `/manage/financial` (计件明细), `/manage/staff` (扫码)
- 计件工资模块与财务管理集成，根据扫码环节记录的数据自动统计员工工资。
- 扫码跟踪模块提供简洁的移动端界面，方便工人在生产线通过扫码更新工序状态。

### 3.6 营销与展示模块

#### 3.6.1 炫图
- **URL:** `/rendering`
- 强大的AI渲染工具，能将2D门窗设计图快速渲染成逼真的3D室内效果图。
- 支持安装合成、门窗擦除、室内装修等多种AI场景，是门店营销的利器。

#### 3.6.2 阳光房与拍照搭配
- **URL:** `/sunnyHouse`, `/customOrder`
- **阳光房:** 提供独立的3D阳光房设计工具，支持模板选择和自定义设计。
- **拍照搭配:** AR功能，允许用户将门窗模型实时叠加在手机拍摄的实景照片上，预览安装效果。

#### 3.6.3 窗型图库
- **URL:** `/windoor-library/index`
- 企业级的窗型模板库，用户可将优秀的设计保存到图库中，供团队复用。
- 支持按分类浏览和搜索。

### 3.7 系统与用户模块

#### 3.7.1 教程中心
- **URL:** `/userGuide`
- 替代了传统的“个人中心”，提供按模块分类的视频教程，帮助用户快速上手软件的各项功能。

#### 3.7.2 系统设置
- **URL:** `/personal-center/account-setting`
- 用户可在此配置个人账号信息（头像、昵称）、修改密码、绑定微信等。
- 管理员还可在此进行子账号管理、查看用户协议等。
