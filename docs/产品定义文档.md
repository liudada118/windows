# “画门窗” (WindoorCraft) 产品定义文档

**版本: 1.0**

**作者: Manus AI**

**日期: 2026年3月1日**

## 1. 产品概述

“画门窗” (WindoorCraft) 是一款专为全球门窗行业的工厂和门店设计的SaaS软件 [6]。它旨在提供一个从设计、报价、渲染到生产管理的全流程解决方案，核心价值在于通过极简的操作界面和强大的后端计算能力，帮助中小型门窗企业实现数字化转型，提升效率并降低成本 [1] [5]。该产品支持中英双语，并可在手机、电脑、平板等多端协同操作，实现数据实时同步 [6]。

根据其母公司新格尔集团的定位，“画门窗”主要面向中小型门窗企业，提供“傻瓜式”的设计拆单体验，是集团庞大工业软件产品矩阵中的重要一环 [4]。

## 2. 核心功能

“画门窗”的产品功能覆盖了门窗业务从前端销售到后端生产的多个关键环节，可以大致分为设计绘图、报价与展示、生产管理三大模块。

### 2.1 设计绘图

设计是软件的入口和核心功能，强调“一横一竖画门窗”的易用性，即使是零基础用户也能快速上手 [7]。

| 功能点 | 详细描述 |
| :--- | :--- |
| **基础绘图** | 通过简单的拖拉拽操作，快速绘制平开、推拉、折叠等常规门窗，以及圆弧、转角等各类异形门窗 [7]。 |
| **组件化设计** | 用户可以通过拖放预设的框架 (Frame)、中梃 (Mullion)、扇 (Sash) 等组件来构建复杂的窗型 [1]。 |
| **自由编辑** | 支持对框架、中梃、五金件等进行自由拖拽、旋转和尺寸调整，实现高度定制化的设计 [1]。 |
| **CAD兼容** | 支持将设计图导出为CAD格式，便于与其他设计软件协同工作 [7]。 |
| **多端同步** | 在手机、平板、电脑上所做的任何设计都会实时同步，方便用户随时随地办公 [6]。 |

### 2.2 报价与展示

围绕设计方案，软件提供了一系列功能，旨在帮助门店提升客户沟通和销售转化的效率。

| 功能点 | 详细描述 |
| :--- | :--- |
| **快速报价** | 根据设计图自动生成详细的报价方案 [6]。 |
| **AI渲染** | 能在10秒内生成设计师级别的逼真效果图，并能将门窗效果合成到客户提供的实景照片中，预览安装效果 [1]。 |
| **3D动画** | 可渲染出门窗的3D动画效果，动态展示开启方式等细节 [6]。 |
| **方案分享** | 支持生成专业的报价单、大样图等PDF文件，并可通过链接直接分享给客户 [1] [7]。 |
| **电子签名** | 客户可以直接在分享的方案上进行电子签名确认，简化订单流程 [7]。 |

### 2.3 生产管理

“画门窗”打通了从门店到工厂的数据流，为后端生产提供了强大的支持。

| 功能点 | 详细描述 |
| :--- | :--- |
| **自动拆单算料** | 门店订单确认后，工厂端可立即完成拆单，自动计算并生成型材和玻璃的下料清单 [6] [7]。 |
| **型材优化** | 内置智能算法，对型材进行优化排版，提高材料利用率，并支持余料管理 [6]。 |
| **报表生成** | 可一键生成门窗下料单、玻璃生产单、型材优化单、成品合格证等多种生产报表 [7]。 |
| **设备对接** | 实现与各类主流CNC数控设备（如双头锯、钻铣加工中心、铣锯一体机等）的无缝对接 [6]。 |
| **生产追溯** | 支持通过手机扫码跟踪订单的生产进度，实现车间无纸化管理 [6]。 |
| **库存管理** | 可根据生产消耗自动扣减仓库物料，实现基础的库存管理功能 [6]。 |

## 3. 技术架构与平台

- **核心技术**: 产品的Web端基于一个名为 `webcc.js` 的HTML5库构建，该库由公司自主研发，基于TypeScript编译，并开放给第三方开发者集成 [2]。
- **平台支持**: 产品为跨平台SaaS应用，覆盖Web浏览器、Windows桌面端 [8]、iOS及Android移动端 [5]，实现了全平台数据互通。

## 4. 市场与商业模式

- **目标用户**: 主要面向全球范围内的中小型门窗工厂和门店 [4] [6]。
- **产品定价**: 采用SaaS订阅模式，按年付费。根据华为云商店的信息，区分为“设计版”和“算料版”，其中“设计版”年费为980元人民币 [7]。
- **销售渠道**: 主要通过官方网站、各大应用商店（如华为云商店、腾讯应用宝、微软应用商店）以及母公司新格尔集团的渠道进行销售 [5] [7] [8]。

## 5. 公司背景

“画门窗”由**无锡窗造软件有限公司**开发和运营 [5] [6]。该公司是**江苏新格尔人居科技有限公司**（ThinkerX）的子公司 [4]。新格尔集团深耕人居行业十余年，拥有强大的研发实力和庞大的客户基础，为“画门窗”提供了坚实的技术和市场支持 [3]。
