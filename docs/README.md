## 文档维护原则

1. **PRD_V5_Complete.md 是唯一的需求来源 (Single Source of Truth)**。所有新需求和变更必须更新到此文档。
2. 任何代码修改后，必须同步更新 `ARCHITECTURE.md`。
3. 对文档库结构进行任何调整后，都必须同步更新此 `README.md` 文件。
