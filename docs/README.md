## CI 校验

项目提供了结构完整性校验脚本，位于 `scripts/check_prd_structure.py`。建议在每次修改 PRD 后运行：

```bash
python3 scripts/check_prd_structure.py
```

该脚本检查 7 项指标：章节编号连续性、子章节归属、无重复编号、结束标记唯一性、结束标记后无多余内容、API 路径一致性、版本号一致性。

## 文档维护原则

1. **PRD_V6_Complete.md 是唯一的需求来源 (Single Source of Truth)**。所有新需求和变更必须直接更新到此文档的正确章节位置。**严禁在文档末尾追加内容。**
2. 修改 PRD 后，必须运行 `scripts/check_prd_structure.py` 校验结构完整性。
3. 任何代码修改后，必须同步更新 `ARCHITECTURE.md`。
4. 对文档库结构进行任何调整后，都必须同步更新此 `README.md` 文件。
