# 画图模块详细交互设计文档

**版本：** 1.0  
**日期：** 2026-03-01  
**负责人：** Manus AI  

---

## 1. 概述

本文档旨在精确定义画门窗软件**画图核心模块**的所有交互细节，作为前端开发的直接依据。目标是复刻一个与原版在交互层面完全一致的画图引擎。

**核心交互理念：**

- **模板驱动**：通过左侧工具箱选择组件模板，然后在画布上放置。
- **选中后修改**：通过单击玻璃区域选中组件，然后在右侧属性面板修改参数。
- **实时联动**：画布上的任何操作（拖拽、修改尺寸）都会实时更新渲染效果和尺寸标注。

---

## 2. 画布布局与渲染

### 2.1 画布结构

画布在垂直方向上分为两个主要区域：

| 区域 | 描述 | 作用 |
|---|---|---|
| **主视图区域** | 占据画布约80%的上方空间 | 显示门窗的正面3D渲染效果和尺寸标注 |
| **平面视图区域** | 占据画布约20%的下方空间 | 显示门窗的水平截面图（俯视图），用于展示型材厚度和内外关系 |

### 2.2 渲染风格

- **型材（外框、中梃、扇框）**：采用带有光影和渐变效果的3D立体渲染，通常为灰色或根据颜色面板设置的颜色。
- **玻璃区域**：采用浅蓝色半透明效果渲染，模拟玻璃材质。
- **尺寸标注**：采用红色细线、红色箭头和红色数字，清晰地标注在主视图和平面视图周围。

### 2.3 组件渲染规则

| 组件 | 渲染特征 |
|---|---|
| **外框/中梃** | 3D立体型材效果，有厚度感。 |
| **固定玻璃** | 浅蓝色半透明填充。 |
| **内开扇** | 在玻璃区域基础上，增加从四角到中心的**对角虚线交叉（X形）**标记，并在扇框顶部标注开启角度（如 "90°"）。 |
| **纱扇** | 在玻璃区域基础上，增加细密的灰色网格线效果。 |

### 2.4 尺寸标注系统

- **层级**：尺寸标注分为两层，第一层为各分段尺寸，第二层为总尺寸。
- **位置**：宽度标注在窗框下方，高度标注在窗框左侧。
- **样式**：红色细线、双向箭头、红色数字。
- **实时性**：任何导致尺寸变化的操作（如拖拽中梃、修改总宽）都会**实时更新**所有相关的尺寸标注数字。

## 3. 核心交互：选中、添加与修改

画图模块的核心交互循环可以概括为：**从左侧选择模板 → 在画布上放置 → 在右侧修改属性**。

### 3.1 组件选中与属性面板 (Select & Modify)

**选中的是“区域”，而不是“型材”**。这是理解交互的关键。

**选中规则：**

| 用户操作 | 目标区域 | 结果 | 右侧面板状态 |
|---|---|---|---|
| **单击** | **玻璃区域** (固定格或扇内) | 该区域被**绿色实线边框**高亮包围 | 自动切换到 **属性 (Properties)** Tab，显示该区域的属性 |
| 单击 | 型材区域 (外框、中梃、扇框) | **不选中**任何组件，取消已有选中 | 切换回 **订单 (Order)** Tab |
| 单击 | 画布空白区域 | **不选中**任何组件，取消已有选中 | 切换回 **订单 (Order)** Tab |

**属性面板 (Properties Panel) 动态变化：**

- **默认状态** (无选中)：右侧面板显示 **订单 (Order)** Tab，包含窗号、总宽、总高、系列相关配置等全局属性。
- **选中状态** (选中玻璃区域)：右侧面板显示 **属性 (Properties)** Tab，其内容根据选中区域的类型动态生成。例如，选中固定格后，会显示“连接方式”、“分框”、“锁定拖动”等操作按钮和属性字段。

### 3.2 左侧工具箱与组件添加 (Add)

左侧工具箱是添加所有画布组件的唯一入口。

**结构：**

- **一级Tab**：按组件类型分为 **框、中梃、平开扇、推拉/折叠、格条、填充物、标线、其他** 等8个主要Tab。
- **二级Tab**：在每个一级Tab内部，可能还有二级Tab对模板进行细分。例如，中梃分为“竖梃”、“横梃”、“组合梃”等。
- **模板列表**：以图标形式展示所有可用的组件模板。

**添加流程：**

1.  **选择模板**：在左侧工具箱中单击一个组件模板图标。
2.  **放置组件**：将鼠标移动到画布的目标区域（通常是玻璃区域），单击以放置组件。

**关键动态行为：**

- **系列决定模板**：左侧工具箱中可用的模板列表（尤其是“常用”和“扇”模板）会根据当前在订单面板选择的**型材系列**（如“平开系列” vs “推拉系列”）动态变化。
- **二级选择菜单**：对于复杂组件（如“扇”），在画布上点击后，可能会弹出一个**子类型选择菜单**，要求用户做进一步的选择（例如，选择单扇、双扇、固定玻璃等）。

## 4. 核心交互操作

### 4.1 拖拽移动中梃

- **操作**：鼠标左键按住中梃的型材区域，并拖动。
- **行为**：
    - 中梃会跟随鼠标实时移动。
    - 两侧的玻璃区域宽度会实时重新计算并调整。
    - 底部的分段尺寸标注数字会实时更新。
- **约束**：可能存在最小宽度限制，防止将一个区域拖拽得过小。

### 4.2 修改尺寸

- **入口**：在右侧 **订单 (Order)** 面板中修改“总宽”、“总高”输入框的值。
- **行为**：
    - 修改总尺寸后，画布上的窗型会按比例缩放或根据内部均分规则重新计算所有分格的尺寸。
    - 所有尺寸标注会实时更新。

### 4.3 删除组件

- **入口**：顶部工具栏的“删除”按钮。
- **行为**：
    1. 先通过单击玻璃区域**选中**一个或多个组件。
    2. 点击“删除”按钮。
    3. 被删除的组件将从画布上移除。如果是中梃被删除，相邻的两个玻璃区域会自动合并为一个。

### 4.4 撤销/恢复 (Undo/Redo)

- **入口**：顶部工具栏的“撤销”和“恢复”按钮。
- **操作栈**：几乎所有对画布内容产生修改的操作都应被记录到撤销/恢复的操作栈中，包括：
    - 添加/删除组件
    - 拖拽中梃
    - 修改尺寸
    - 修改属性（如颜色、型材）

### 4.5 画布缩放与平移

- **缩放**：通过鼠标滚轮实现。向前滚动放大，向后滚动缩小。缩放以鼠标指针位置为中心。
- **平移**：按住鼠标中键或空格键，然后拖动鼠标，可以平移整个画布。
