# 画门窗软件功能参考文档

## 1. 引言

本文档旨在根据对“画门窗”软件的详细调研，整理并输出一份结构化的功能参考文档，为后续的产品设计和开发提供清晰、可执行的功能依据。调研内容基于对软件各项功能的实际操作和观察，涵盖了从基本界面布局到核心绘图、渲染功能的方方面面。

## 2. 主界面布局

画门窗软件的主界面分为三大区域：顶部工具栏、左侧组件/工具面板、右侧属性/订单面板，以及中心的绘图画布。

| 区域 | 主要功能 |
| --- | --- |
| **顶部工具栏** | 提供撤销、恢复、删除、清除、保存、渲染等全局操作。 |
| **左侧面板** | 核心组件和工具选择区，包含窗框、中梃、平开扇、推拉/折叠、格条、填充物、标线、其他工具等。 |
| **右侧面板** | 信息展示与编辑区，包含订单信息、颜色配置、选中组件的属性、图纸预览、报价等。 |
| **绘图画布** | 用户进行门窗设计和绘制的核心工作区域。 |

## 3. 左侧面板：组件与工具

左侧面板是门窗设计的核心功能区，分为组件模板和辅助工具两大部分。

### 3.1 框 (F)

“框”是构建门窗的基础，软件提供了极为丰富的框模板库，远超常规的矩形框。

- **模板数量**：约60+种
- **支持类型**：
    - **基础矩形**：单框、二等分、三等分等。
    - **异形框**：支持圆形、半圆、弧形、拱形、三角形、梯形、菱形及各种不规则多边形。
    - **组合框**：支持带弧顶、三角顶的矩形框等复杂组合。
    - **特殊框**：转角框、拼框等。
- **关键特性**：
    - 拥有独立的“异形框”子Tab，表明支持用户自由绘制或编辑异形轮廓。
    - 模板化的设计大大提高了异形窗的设计效率。

### 3.2 中梃 (V)

中梃用于在窗框内部分割区域，同样提供了丰富的模板。

- **模板数量**：约30+种
- **支持类型**：
    - **常规分割**：横向、竖向、井字、田字、T形等。
    - **等分/不等分**：支持二/三/四等分，以及自定义不等分比例。
    - **组合分割**：支持复杂的组合分割样式。
- **关键特性**：
    - 拥有“异形中梃”子Tab，推测支持曲线或斜线中梃。
    - 模板可以被拖拽到画布中的任意玻璃区域，自动完成分割。

### 3.3 平开扇 (S)

平开扇是门窗中的活动单元，软件支持多种开启方式和组合形式。

- **模板数量**：约30+种
- **支持类型**：
    - **开启方式**：左开、右开、上悬、对开。
    - **组合形式**：支持单扇、双扇、三扇、四扇及多扇组合，并支持平开与上悬的混合组合。
- **关键特性**：
    - 模板图标清晰地用三角形标记了扇的开启方向。
    - 支持将平开扇模板拖拽入任意由框或中梃分割出的玻璃区域。

### 3.4 推拉/折叠 (D)

推拉与折叠功能被整合在同一个Tab下，提供多种轨道和扇组合模板。

- **模板数量**：约12+种
- **支持类型**：推拉门、W型推拉、折叠玻扇。
- **关键特性**：
    - 快捷键统一为“D”。
    - 模板覆盖了常见的双轨、三轨以及多扇推拉/折叠组合。

### 3.5 格条

格条用于在玻璃内部添加装饰性或结构性的分割线条。

- **模板数量**：约40+种
- **支持类型**：
    - **直线型**：平行横/竖格条、井字、田字、菱形、斜线等。
    - **曲线型**：弧形、圆形、拱形、S形等。
    - **图案型**：支持复杂的中式或欧式装饰格条图案。
- **关键特性**：
    - 格条模板可以直接拖拽到任意玻璃区域上。
    - 丰富的模板库可以满足多样化的装饰需求。

### 3.6 填充物

填充物功能允许用户将玻璃替换为其他材质或组件。

- **模板数量**：约8种
- **支持类型**：纱窗、百叶窗、板材、用户自定义图片等。
- **关键特性**：
    - 拥有独立的图层管理功能，可以管理不同填充物的层级关系。
    - 极大增强了门窗设计的灵活性和真实感。

### 3.7 标线

标线工具用于在图纸上添加各种标注信息。

- **工具类型**：约9种
- **支持功能**：
    - **尺寸标注**：线段、折线、对角线、圆弧标注。
    - **符号标注**：箭头标注。
    - **文字标注**：添加自定义文字说明。
    - **辅助线**：垂直、水平标注线。
    - **编辑工具**：橡皮擦可用于删除标注。
- **关键特性**：
    - 标注线的端点（红色圆点）可以被拖拽，以精确控制标注位置。

### 3.8 其他

包含一系列绘图辅助工具。

- **工具类型**：约6种
- **支持功能**：网格、复制、分割、尺寸、对齐、分布等。
- **关键特性**：这些是通用的CAD辅助功能，用于提高绘图的精确性和效率。

## 4. 右侧面板：信息与属性

右侧面板是信息管理和细节调整的核心区域。

### 4.1 订单

“订单”Tab集中管理与当前门窗相关的所有业务信息。

- **核心字段**：总宽、总高、窗号、樽数、单价、纱网、外色、内色、玻璃、五金、位置、备注等。
- **关键功能**：
    - **编辑洞口**：可以单独编辑墙体洞口的尺寸。
    - **单位切换**：支持在毫米、厘米、米之间切换截图单位。

### 4.2 颜色

颜色面板提供了强大而灵活的配色方案。

- **颜色类型**：支持“纯色”和“木纹”两大类。
- **色板**：
    - **纯色**：提供约20+种预设颜色，并支持通过色板自定义任意颜色。
    - **木纹**：提供约8种常见木纹贴图，如巴西柚木、红橡、水曲柳等。
- **关键功能**：
    - **独立设置**：支持“全选”对所有框架应用颜色，也支持对“压线颜色”进行单独设置。
    - **颜色库管理**：用户可以新增和管理自己的颜色库。

### 4.3 属性

当选中画布上的某个组件（如玻璃、扇）时，此面板会显示其详细属性。

- **玻璃属性示例**：
    - **磨砂**：开关，控制是否应用磨砂效果。
    - **玻璃颜色**：设置玻璃的颜色。
    - **嵌入百叶**：开关，控制是否在玻璃中嵌入百叶。
    - **玻璃规格**：下拉选择不同的玻璃厚度或类型。

## 5. 绘图与渲染

软件的绘图和渲染能力是其核心价值所在，展现了高度的真实感和细节表现力。

### 5.1 画布渲染

- **3D立体效果**：所有窗框、扇、把手等组件都以带有光影和渐变色的3D效果渲染，立体感强。
- **材质表现**：
    - **框架**：四角呈现逼真的45度斜切角效果。
    - **玻璃**：渲染为半透明效果，颜色可调。
    - **纱窗**：以网格图案清晰地表现出来。
- **细节标注**：
    - **尺寸**：所有尺寸都以红色标注线清晰地显示在画布上，并实时更新。
    - **开启方式**：平开扇内部会用交叉虚线标记开启方向。
    - **室内/外**：画布底部明确标识室内外朝向，并提供截面图预览。

### 5.2 图纸模板

点击画布左上角的齿轮图标，可以访问一个强大的图纸模板库。

- **模板内容**：库中包含了大量预设的完整门窗图纸，如“内外砂灰 免打胶排水孔 分体”、“外深喖内砂灰 内开内倒”等，覆盖了平开、推拉、组合窗、中式格条窗等多种复杂样式。
- **一键应用**：用户可以点击任意模板，其完整的结构、尺寸、颜色、标注等会立刻应用到当前画布上，作为设计的起点或参考。
- **关键价值**：极大地降低了复杂门窗设计的门槛，为用户提供了丰富的设计灵感和可复用的方案。

## 6. 结论

“画门窗”软件是一款功能高度集成且细节表现出色的专业门窗设计工具。其核心优势在于：

- **丰富的模板库**：无论是窗框、中梃还是格条，都提供了远超常规的模板选择，尤其是在异形窗和复杂组合窗方面表现突出。
- **强大的可视化与渲染能力**：逼真的3D渲染、细腻的材质表现和清晰的细节标注，为用户提供了所见即所得的设计体验。
- **高度的灵活性与自定义能力**：从颜色库管理到自定义异形框，再到丰富的组件属性设置，软件在标准化与个性化之间取得了很好的平衡。

本文档的功能梳理可为我方产品的迭代和优化提供重要参考，尤其是在扩展窗型支持、提升渲染质量和增强设计灵活性等方面。
